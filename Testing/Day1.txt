What is Java?
	Java is a class-based, object-oriented programming language

variable:
	name of datatype.

Primitive data types:
	 boolean (true, false),
	 char (a,b,c....z),
	 int (1,2,3....), 
	 short (1,2,3....), 
	 byte (1,2,3....), 
	 long (1,2,3....), 
	 float (1.0,1.2,1.3....N), 
	 double(1,2,3......N).

Data Type	Description
	byte	Stores whole numbers from -128 to 127
	short	Stores whole numbers from -32,768 to 32,767
	int		Stores whole numbers from -2,147,483,648 to 2,147,483,647
	long	Stores whole numbers from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
	float	Stores fractional numbers. Sufficient for storing 6 to 7 decimal digits
	double	Stores fractional numbers. Sufficient for storing 15 to 16 decimal digits
	boolean	Stores true or false values
	char	Stores a single character/letter or ASCII values

Non Primitive data types:
	 String, Array, etc.


wrapper classes:
	 Integer, Character, Boolean

Methods and functions

Class - {data + methods}

Return type:
	In Java programming, the return statement is used for returning a value when the execution of the block is completed.

constructor:
	block of code that initialize the newly created object.

Conditions:
	1. Constructor name must be the same as its class name.
	2. Must not have explicit return type.
	3. cannot abstract, static, final and synchronized.

Constructor types:
	1. default:
		when we don't provide any constructor, compiler will provide one. we cannot see naked eye. used to provide the default value.

	2. No arguments constructor:(User define constructor)
		No arguments will be there it will be allow us to write logic.

	3. Parameterized:(User define constructor)
		It has arguments (parameters).

Constructor overloading.

Static:
	1. can be used with class, Variable, method and block of code.
	2. belong to the class instead of specific instance, this means if make a member static, you can access without object.
